# Doufu
一个简易的曲谱同步视频制作工具

## 使用前提
安装 `ffmpeg` 并添加到环境变量中

## 快捷键
1. 播放/暂停音频: Space
2. 将选择的区域设置为当前音频时间: Enter
3. 新建区域: n
4. 删除区域: d
5. 后退5s/前进5s: <- / ->
6. 上一个/下一个区域: ↑ / ↓
